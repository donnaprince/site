    Maze-Tastic!

    August 29, 2017 to September 7, 2017
    Group Project

    My very first project in my first computer science course was utilizing the Scratch platform. The overall objective of the project was to accumulate knowledge about variable roles and receive an overview of engineering by creating a game in Scratch. Our group chose to create a game titled Maze-Tastic!. We organized our brainstorming using a flowchart and tracked our progress on a document. In this project, I was tasked with the documentation and commenting on our code. One skill assimilated from the project was how to document all the information accurately and in an orderly fashion. Another strength, I was able to garner was using Scratch in different ways to code the same function. In the project, our team was able to produce a very refined game due to our teamwork.
